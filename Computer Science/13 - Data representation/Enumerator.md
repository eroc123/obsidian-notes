- user defined 
- non - composite
- maps a **set** of names to a specific number
- is **ordered** and **countable**, with finite values in the set
- 
- e.g. for a set called months

| name      | enumerator |

| jan       | 1          |
| feb       | 2         |
| march     | 3          |
| april     | 4          |
| may       | 5          |
| june      | 6          |
| july      | 7          |
| august    | 8          |
| september | 9          |
| october   | 10         |
| november  | 11         |
| december  | 12         |
